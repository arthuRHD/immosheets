# Security Policy

## Supported Versions

Since it's a product under development, we have breaking changes and we don't maintain early releases before 1.1.x

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

If you find a vulnerability, please contact me at <arthur.richard@immosheets.io> or create an Issue on
this repository.
